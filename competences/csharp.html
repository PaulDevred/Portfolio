Ma Vision du C#

Le C# est un langage que j'apprécie pour son équilibre entre performance, clarté et puissance. Il me permet de concevoir des applications robustes, maintenables et orientées métier, aussi bien dans des contextes professionnels (API, intégrations, automatisations) que dans des projets d'étude.

Dans mon alternance chez Iothink-Solutions, spécialisée dans l'IoT, cette compétence a été essentielle. J'ai pu exploiter pleinement la richesse de l'écosystème .NET pour concevoir des services fiables et sécurisés, exploitables en production.

Ce que je développe en C#

1. Intégration de données externes (GRDF, Actility)

Je conçois plusieurs outils en C# permettant d'intégrer des données d'API tierces, notamment celles de GRDF et Actility. Ces outils permettent de rendre les données exploitables par nos clients via les systèmes internes de l'entreprise.

Ce développement vise à positionner notre plateforme comme tiers autorisé dans la gestion de données énergétiques : un rôle qui demande une sécurité et une fiabilité à toute épreuve.

Fonctionnalités intégrées

  • Sécurisation des flux de données
  • Traçabilité des autorisations
  • Intégration API complexes
  • Module de gestion du consentement automatisé
  • Intégration DocuSign
  • Respect des règles de confidentialité

2. Synchronisation entre logiciels internes

Je développe un projet en C# destiné à lier deux logiciels internes afin de synchroniser leurs données en temps réel. Ce projet combine API REST et webhooks pour détecter et propager les changements d'état.

Les données reçues sont ensuite traitées et interprétées selon une logique métier spécifique. J'ajoute une interface de configuration afin de permettre aux équipes d'adapter le comportement du programme à leurs besoins sans intervention technique.

3. Darwin Simulator (Unity)

Dans le cadre d'un projet universitaire nommé Darwin Simulator, j'ai exploité le C# au sein d'Unity pour simuler un écosystème évolutif. J'y ai implémenté un algorithme génétique NEAT (NeuroEvolution of Augmenting Topologies) permettant à des entités virtuelles d'apprendre et d'adapter leur comportement.

Ce projet m'a permis de consolider ma maîtrise des concepts orientés objet et de la programmation algorithmique avancée.

Mon niveau en C#

Je considère aujourd'hui avoir une bonne maîtrise du langage C# et de son écosystème .NET. J'ai acquis une solide expérience dans la conception de code clair, modulaire et documenté, capable d'évoluer sur la durée.

check_circle Points forts

  • Conception d'architectures logiques cohérentes
  • Interaction avec des API complexes
  • Anticipation des contraintes de sécurité et de performance
  • Application des principes du clean code et design patterns
  • Utilisation régulière de xUnit pour les tests unitaires

trending_up Axes d'amélioration

  • Exploitation des outils CI/CD dans un contexte .NET
  • Maîtrise de la gestion de la mémoire et performances bas niveau (Garbage Collector, Span<T>, structs immuables)
  • Tests de charge (BenchmarkDotNet)
  • Stratégies de test dans les architectures microservices

Ma maîtrise du C# est particulièrement adaptée aux contextes de développement backend, d'intégration API et de logique métier complexe. Je suis moins à l'aise sur les aspects de performance bas niveau et d'optimisation mémoire avancée, mais ces compétences font partie de mon plan de progression.
